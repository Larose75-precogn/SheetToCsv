# SheetToCsv Premium

Un add-on Google Sheets qui permet d'exporter facilement vos feuilles de calcul au format CSV.

## 🚀 Fonctionnalités

- Export de plusieurs feuilles en un seul CSV
- Ajout d'une colonne avec le nom de la feuille d'origine
- Format CSV standard (séparateur ;)
- Version Premium avec export illimité

## 📋 Versions

| Version | Fonctionnalités |
| :--- | :--- |
| **Gratuite (Essai)** | 2 feuilles maximum, 7 jours d'essai |
| **Premium** | Feuilles illimitées, export en un clic |

## 💳 Paiement
